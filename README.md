# UTFSM_2025S2-IA-
Este repositorio implementa un algoritmo de Simulated Annealing (SA) para resolver el Rotating Workforce Scheduling Problem (RWSP), un problema NP-completo de planificación de turnos con restricciones de cobertura, bloques, secuencias prohibidas y fines de semana libres.


Proyecto RWSP-SA
│── main.cpp
│── *.cpp / *.hpp          # Código fuente del algoritmo
│── Makefile
│── parametros.txt         # Configuraciones de SA a probar
│── metricas.csv           # Se genera automáticamente
│── Resultados/            # Soluciones generadas
│── Instancias/            # Archivos .txt del RWSP
