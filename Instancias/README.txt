Instancias para RWSP

Separadas por tamaños de N (cantidad de trabajadores) y cada carpeta NX/ incluye instancias solo con N = X, no mayor ni menor.

Carpeta NPlus incluye algunas instancias extras, las cuales tienen tamaño N mayor a 50.

Instancias siguen el formato indicando en la presentación del problema (Presentacion_RWSP_Extended_ver_.pdf).

Nombre de los archivos de salida debe ser <nombre_instancia>_<seed>.txt o <nombre_instancia>.txt según corresponda.
